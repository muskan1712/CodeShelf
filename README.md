# 📚 CodeShelf

CodeShelf is a full-stack MERN application that serves as a digital platform for discovering and managing programming learning resources. It provides curated resources across multiple computer science domains such as Data Structures & Algorithms, DBMS, Operating Systems, and Artificial Intelligence, helping students and developers learn efficiently.

## 🚀 Features

- Browse programming resources by category
- Explore DSA, DBMS, OS, AI, and other technical topics
- Responsive and user-friendly interface
- Resource management through REST APIs
- Cart functionality for saving resources
- Dynamic content rendering using React
- State management with Redux Toolkit

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Redux Toolkit
- Vite

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/muskan1712/CodeShelf.git
cd CodeShelf
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
